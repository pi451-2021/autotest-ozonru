\section{Общее описание продукта}

PetClinic --- это демонстрационный проект, в котором представлено 
обобщенное 
использование Spring Boot, Spring MVC и Spring Data. PetClinic имеет 
давнюю историю, восходящую к началу Spring Framework. Он зародился как 
демонстрация почти всех возможностей, которые предоставлял Spring. В 
наши дни это лишь небольшая часть всех его возможностей

\textbf{Функциональные возможности:}

\begin{itemize}
\item Просмотр информации о ветеринарах и их специальностях
\item Просмотр информации о владельцах питомцев
\item Изменение информации о владельцах питомцев
\item Добавление новых владельцев
\item Просмотр информации о питомцах
\item Изменение информации о питомцах
\item Добавление новых питомцев
\item Просмотр историй посещений питомцев
\item Добавление записи о новых посещениях питомцев   
\end{itemize}
